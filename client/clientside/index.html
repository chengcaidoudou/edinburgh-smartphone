SSD Model

Portfolio Optimization with Second-order Stochastic Dominance

  • Stock Symbols: A stock symbol should be filled in each textbox. The stock symbol must be valid and exist on Google Finance.
  • Benchmark: To be filled with a stock index symbol, so that the computed optimal portfoilo would perform better than it stochastically. The index symbol must be valid and exist on Yahoo Finance.
  • Data Range: The start and end date for the stock pricing data retrieval. Weekly stock pricing data will be retrieved.
  • Add Symbol,Del Symbol buttons are used to add or del a textbox on the stock symbols field.
  • Solve button is used to send the request to the web service which is powered by SML and OOPS.

MAD Model

Portfolio Optimization with Mean Absolute Derivation(MAD) as risk measurement

  • Stock Symbols: In each textbox, a stock symbol should be filled. The stock symbol must be valid and exist on Google Finance.
  • Return Ratio: The miminal return ratio that you want for your optimized portfolio.
  • Data Range: The start and end date for the stock pricing data retrieval. Weekly stock pricing data will be retrieved.
  • Add Symbol,Del Symbol buttons are used to add or del a textbox on the stock symbols field.
  • Solve button is used to send the request to the web service which is powered by SML and OOPS.
